Welcome to HARK’s documentation!¶

Contents:

HARKutilities This module contains a number of utilities, including the code that implements agent expectations, utility functions (and their derivatives), manipulation of discrete approximations, and basic plotting tools.
HARKsimulation A collection of functions used for generating simulated data and shocks.
HARKparallel This is a very minimal module for an early version of multithreading in HARK.
HARKinterpolation This module contains custom interpolation methods for representing approximations to functions.
HARKestimation This module contains functions useful for estimating structural models, including optimization methods and bootstrapping tools.
HARKcore This is the missing description for HARKcore.
